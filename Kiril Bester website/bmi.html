The User can enter their Height(cm) and their Weight(kg).

BMI Formula: weight(kg)/[height(m) x height(m)]

Convert meters(m) to centimeters(cm): meter(m)/100cm

Weight Class

These tables are important, because it differentiates between each of the weight classes. Determining on the BMI score that you received, you will see in which weight class you fall under.

BMI Weight Status
Less than 18.5 Underweight
18.6-24.9 Normal
25.0-29.9 Just Overweight
30.0-34.9 Obese class 1
35.0-39.9 Obese class 2
More than 40.0 Obese class 3
